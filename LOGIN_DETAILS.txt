================================================================================
                        API ENDPOINTS
================================================================================

POST /api/auth/signin
  Body: { email: string, password: string }
  Returns: User object with session cookie

POST /api/auth/signup
  Body: { email: string, password: string, name: string }
  Returns: New user object with session cookie

POST /api/auth/signout
  No body required
  Clears session cookie

GET /api/auth/session
  Returns: Current user object or null

POST /api/admin/upload
  Body: FormData with file and category
  Returns: { accountsCount: number }
  Requires: Admin role

================================================================================
                        TROUBLESHOOTING
================================================================================

CAN'T LOGIN
  ✓ Check email spelling (case-insensitive)
  ✓ Verify password (case-sensitive): Admin@2026 or User@2026
  ✓ Clear browser cookies: Settings → Clear browsing data
  ✓ Try incognito mode
  ✓ Check console for errors (F12)

CAN'T ACCESS ADMIN
  ✓ Must login with admin account first
  ✓ Non-admin accounts cannot access /admin
  ✓ Try: admin@verifiedstore.com / Admin@2026

UPLOAD NOT WORKING
  ✓ File must be .txt format only
  ✓ File cannot be empty
  ✓ One account per line required
  ✓ Select category before upload
  ✓ Check file size (under 50MB)

SESSION EXPIRED
  ✓ Session lasts 30 days
  ✓ Clear cookies to reset: Ctrl+Shift+Delete
  ✓ Login again to create new session
  ✓ Check if cookies are enabled

CORRECT .txt FILE FORMAT

UPLOAD PROCESS
  1. Create .txt file (notepad, VS Code, etc.)
  2. Add accounts (one per line)
  3. Save file with .txt extension
  4. Login as admin
  5. Go to /admin
  6. Select category
  7. Upload file
  8. View in history
