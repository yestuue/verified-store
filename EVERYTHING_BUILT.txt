PART 10: GETTING STARTED
================================================================================

1. SET UP DATABASE
   • Go to neon.tech and create project
   • Copy DATABASE_URL
   • Add to .env.local
   • Run: psql $DATABASE_URL -f scripts/init-db.sql

2. INSTALL DEPENDENCIES
   • pnpm install

3. CREATE ADMIN USER
   • Register account via Sign Up
   • Update in database: UPDATE users SET is_admin = true WHERE email = 'your@email.com';

4. RUN DEV SERVER
   • pnpm dev
   • Open http://localhost:3000

5. TEST FEATURES
   • Browse products
   • Sign up / Sign in
   • Checkout process
   • Visit /admin for file uploads

6. DEPLOY
   • Connect to Vercel
   • Add DATABASE_URL env var
   • Deploy with git push
